# PPT 素材库插件 — 同事安装指南

> 500+ 图形、图标、图表元素 · 一键插入 · 离线可用

---

## 方案一：网络共享安装（推荐，局域网内同事）

> 适用于：和你在同一局域网 / 能通过 VPN 访问内网的同事

只需在 PowerPoint 信任中心添加一个网络共享路径，无需下载任何文件。

### 步骤

1. 打开 PowerPoint → **文件** → **选项** → **信任中心** → **信任中心设置**
2. 左侧选择 **「受信任的加载项目录」**
3. 在 **「目录 URL」** 输入框中填入服务器共享路径：

   ```
   \\LAPTOP-1MVV1RA5\add-in
   ```

   > ⚠️ 请根据实际情况替换为你收到的共享路径

4. 点击 **「添加目录」**
5. 勾选 ✅ **「显示在菜单中」**
6. 点击 **确定** → 再点 **确定** 关闭选项窗口
7. **完全关闭 PowerPoint 再重新打开**（必须重启！）
8. 打开任意 PPT 文件
9. **插入** → **获取加载项**（或 **加载项**）
10. 点击左上角 **「共享文件夹」** 选项卡
11. 找到 **「PPT Asset Library」**，点击 **「添加」** 🎉

### 注意事项

- 共享路径的电脑/服务器必须**开机在线**，否则首次安装时会找不到
- 安装成功后，插件内容从 GitHub Pages 加载，**不再依赖共享路径**
- 第一次打开插件需要联网，之后离线也能用（自动缓存）

## 方案二：手动侧载安装（适用于所有场景）

> 适用于：不在局域网内 / 无法访问网络共享的同事

### Windows

1. 获取 `manifest-gh-pages.xml` 文件（找同事要，或从 GitHub 下载）
2. 在「文档」文件夹里新建一个 **OfficeAddins** 文件夹，完整路径类似：

   ```
   C:\Users\你的用户名\Documents\OfficeAddins
   ```

3. 把 `manifest-gh-pages.xml` 放进这个文件夹
4. 打开 PowerPoint → **文件** → **选项** → **信任中心** → **信任中心设置**
5. 左侧选 **「受信任的加载项目录」**
6. 在 **「目录 URL」** 中填入上面的文件夹路径（注意把「你的用户名」替换为实际用户名）：

   ```
   C:\Users\你的用户名\Documents\OfficeAddins
   ```

7. 点击 **「添加目录」** → 勾选 ✅ **「显示在菜单中」** → 确定
8. **完全关闭 PowerPoint 再重新打开**
9. **插入** → **获取加载项** → **共享文件夹** 选项卡
10. 找到 **「PPT Asset Library」**，点击 **「添加」** 🎉

### Mac

1. 获取 `manifest-gh-pages.xml` 文件
2. 打开 **访达（Finder）**
3. 按快捷键 **Command + Shift + G**（前往文件夹）
4. 粘贴以下路径，点击「前往」：

   ```
   ~/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef
   ```

5. 如果 `wef` 文件夹不存在，就**新建一个**叫 `wef` 的文件夹
6. 把 `manifest-gh-pages.xml` 复制到这个 wef 文件夹里
7. **完全退出 PowerPoint**（Command + Q）
8. 重新打开 PowerPoint，打开任意 PPT
9. **插入** → **加载项** 下拉 ▼ → **我的加载项**
10. 在 **「开发人员加载项」** 中找到 **「PPT Asset Library」** 🎉

## 使用方法

1. 点击顶部 **「开始（Home）」** 选项卡中的 **「Asset Library」** 按钮
2. 右侧弹出素材库面板
3. 浏览分类 / 搜索关键词
4. 点击素材缩略图 → **「插入」**，素材即插入当前幻灯片

## 常见问题

**Q: 安装后在哪里找到插件？**
- Windows：**插入** → **获取加载项** → **共享文件夹** 选项卡
- Mac：**插入** → **加载项** → **我的加载项** → **开发人员加载项**

**Q:「共享文件夹」选项卡里没有插件？**
- 检查是否已勾选「显示在菜单中」
- 确认已**完全关闭并重启**了 PowerPoint（不是关闭文件，是退出程序再打开）
- 如果用的是网络共享方案，确认共享路径的电脑/服务器在线

**Q: 第一次打开很慢？**
- 第一次需要从网络加载素材资源（约 5MB），会自动缓存到本地，之后会很快

**Q: 离线能用吗？**
- 可以！第一次联网使用后，所有资源会自动缓存，之后离线也能正常使用（需要较新版 Office 365）

**Q: 插件会自动更新吗？**
- 会！有网络时会自动后台更新缓存，你始终用的是最新版本

**Q: 怎么卸载？**
- Windows：信任中心 → 受信任的加载项目录 → 选中对应目录 → 删除
- Mac：删除 wef 文件夹里的 xml 文件，重启 PowerPoint
